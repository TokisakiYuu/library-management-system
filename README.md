# 图书管理系统

***(仅作面试用)*** 以快速实现要求的功能为主，用时半小时左右开发完成。

### 功能列表

- 添加图书（图书属性包括: 书名、作者、分类(数组)、评分）
- 更新单个图书的属性
- 查看全部图书
- 根据书名进行模糊查找
- 根据分类进行查找

## 技术栈

### 前端

- next.js
- swr
- typescript

### 后端

- nodejs
- express
- faker.js
- typescript
- 内存数据库

## 如何运行

### 根目录下 `client`目录和 `server`目录夹分别安装好依赖

```bash
pnpm install
```

npm也可以

### `client`目录下执行启动命令

```bash
npm run dev
```

### 打开一个新终端，在 `server`目录下执行启动命令

```bash
npm run start
```
